1-Las versiones de java script han ido evolucionando a lo largo del tiempo a la par de los navegadores, su lenguaje también avanzo y incremento sus capacidades, cuando se desarrola js se depende de la plataforma de ejecución y su compatibilidad o no con alternativas modernas del lenguaje, con la alternativa de este lenguaje que ah evolucionado en 2 vertientes:

  • El lenguaje incorpora operadores, estructura de control, reglas de sistaxis.
  • Los navegadores incorporan instrucciones para manipular elementos modernos de la pagina.
Las versiones son:
Javascript1.1: Se diseño con la llegada de los navegadores 3.0,implementa el tratamiento de imágenes de manera dinámica y la creación de arrays.
Javascript1.2: Versión de los navegadores 4.0, su desventaja es ser distinta en Microsoft y Netscape,ambos crecieron de diferente manera y luchaban por el mercado.
Javascript1.3: Implementada por los navegadores 5.0, esta versión a limado diferencias y asperesas entre los dos navegadores.
Javascript1.5: Versión actual implementada en netscape 6.
Javascript 5.5: Presentada por Microsoft para ser utilizada en nuestros navegadores.
Javascript ECMAScript 5: se publica esta versión en 2009 y funciona en todos los navegadores del mercado.
Javascript ECMAScript6: esta versión es lanzada en 2015, se puede usar en todos los navegadores excepto en internet Explorer, es usada por todos los profesionales, ya que es muy eficiente, y de ser necesario se puede traducir el código para soporte a viejos navegadores.

2-Podemos agregar js a html de dos diferentes modos:
1ra forma: Esta forma es tanto usable en el body como en el head.

-Depende de donde pongas este código, será como va a cargar tu página. Si bien es más aconsejable colocarlo en el head para evitar molestar el código de html, hacerlo en el body, lo hará más rápido.

2da forma: Linkeando el archivo js: esta forma se usa en el head, esta es de las mejores y más utilizadas formas, ya que, podemos reciclar el código de js de ser necesario.

Para usar esta forma, el archivo js debe estar en el mismo directorio que el html.
